## Online point cloud georeferencing tool

GeoRef Cloud is an online tool to georeference, reproject and merge point clouds (LAS, LAZ, EPSG, WGS84, UTM, Lambert).

👉 https://landing.georef.cloud

---

## What is GeoRef Cloud?

GeoRef Cloud allows you to process point clouds directly in your browser, without installing any software.

It is designed for geospatial professionals working with LiDAR and drone photogrammetry data.

---

## Features

- Georeference point clouds (LAS, LAZ)
- Reproject using EPSG coordinate systems
- Convert to WGS84 (EPSG:4326), UTM, Lambert 93, RGF93 CC
- GCP-based georeferencing (ground control points)
- Merge multiple point clouds into a single dataset
- 100% online – no installation required

---

## How it works

1. Upload your LAS or LAZ file  
2. Select the source coordinate system  
3. Choose the target EPSG system (WGS84, UTM, Lambert, etc.)  
4. Add GCP points if needed  
5. Run the transformation  
6. Download the processed point cloud
