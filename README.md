# GeoRef Cloud

GeoRef Cloud is an online tool to georeference, reproject and merge point clouds.

👉 https://landing.georef.cloud

---

## Features

- Georeference point clouds (LAS, LAZ)
- Reproject using EPSG (WGS84, UTM, Lambert)
- Merge multiple point clouds
- GCP-based georeferencing
- 100% online (no installation)

---

## Use cases

- Surveyors and geomatics engineers
- Drone photogrammetry workflows
- LiDAR processing
- Coordinate system transformation
